## yatzy

요트다이스(Yahtzee / Yacht) 점수판을 웹으로 구현한 프로젝트입니다.  
실제 주사위를 굴린 뒤, 결과 점수만 이 페이지에 입력해서 사용할 수 있습니다.

### 주요 기능

- **React 기반 점수판**: Vite + React + TypeScript로 구현된 단일 페이지 앱
- **플레이어 여러 명 지원**: 최대 4명까지 칼럼을 추가해 동시에 기록 가능
- **자동 합계/보너스 계산**:
  - 상단(1~6) 합계 자동 계산
  - 63점 이상일 때 +35점 보너스 자동 반영
  - 하단 합계 및 최종 총점 자동 계산
- **점수 초기화 & 이름 변경**:
  - 한 번에 모든 점수 리셋
  - 각 플레이어 이름 직접 수정 가능
- **반응형 UI**:
  - PC, 태블릿, 모바일에서 모두 사용 가능
  - 작은 화면에서는 가로 스크롤 테이블 형태로 자연스럽게 동작

### 기술 스택

- **프론트엔드**: React, TypeScript
- **번들러/개발 서버**: Vite

### 실행 방법

1. **의존성 설치**

   ```bash
   cd yatzy-web
   npm install
   ```

2. **개발 서버 실행**

   ```bash
   npm run dev
   ```

   브라우저에서 `http://localhost:5173` (Vite 기본 포트 기준)에 접속하면 점수판을 확인할 수 있습니다.

3. **프로덕션 빌드**

   ```bash
   npm run build
   ```

   `dist` 폴더에 정적 파일이 생성됩니다.

### 폴더 구조(요약)

- **루트**
  - `README.md`: 프로젝트 소개 및 실행 방법
  - `yatzy-web/`: Vite + React 앱 소스
- **`yatzy-web/src`**
  - `App.tsx`: 요트다이스 점수판 메인 컴포넌트
  - `main.tsx`: React 엔트리포인트
  - `style.css`: 전체 레이아웃 및 반응형 스타일

### 사용 방법 안내

- **주사위는 실제로 손으로 굴립니다.**
- 각 라운드마다 나온 조합에 맞게 해당 카테고리의 칸에 점수를 입력하세요.
- 점수는 자유롭게 입력할 수 있으므로, 집/모임마다 사용하는 세부 규칙에 맞춰 점수 기준만 조정해서 사용하면 됩니다.
- 상단 합계/보너스, 하단 합계, 총점은 자동으로 계산됩니다.

### 노트

- 현재 Node 22.11.0 환경에서도 빌드되지만, Vite 7 기준으로는 **Node 20.19+ 또는 22.12+**를 권장합니다.
